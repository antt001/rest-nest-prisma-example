# Employee Service REST Example

## Description

This is an employee service REST API implementation exampleusing NestJS, Prisma and Sqlite.
 - The service will run on port 3000.
 - Swagger is available at http://localhost:3000/api

This is a simple example of how to use NestJS, Prisma and Postgres or SQLite. It is not a production ready application.

## Installation

```bash
$ yarn install
```
Prisma client should be generated by the install, but if you run in any prisma client related error, run the following command:
```bash
$ npx prisma generate
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
